# Cognifyz Web Development Internship - Task 4

## Dynamic Form Validation and Interactive UI

A responsive and interactive registration form built using Node.js, Express.js, EJS, Bootstrap, HTML, CSS, and JavaScript.

---

## Features

- Responsive registration form
- Password strength checker
- Show/Hide password feature
- Client-side validation
- Dynamic welcome message
- Bootstrap responsive UI
- EJS templating
- Express.js integration

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Node.js
- Express.js
- EJS

## Installation

### Clone Repository

```bash
git clone https://github.com/DiyaSharma-tech/cognifyz-webdev-task4.git
```

### Install Dependencies

```bash
npm install
```

### Run Server

```bash
node app.js
```

### Open Browser

```bash
http://localhost:3000
```
